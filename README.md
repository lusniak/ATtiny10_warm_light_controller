ATtiny10 is a little (1,6mm/2,9mm) microcontroller from Atmel.
It consists only 6 pins in total, 3 of which are IO (4 if we reconfigure reset pin).
It has 1kB of flash memory and 32B of RAM. This amount cannot contain Arduino framework or any other more extensive HILs.
Direct register manipulation is required.
